
### 制作动态提示输入框

```html
<body>
<input id="i1" type="text" value="请输入关键字" onfocus="Focus();" onblur="Blur()">
<script>
    function Focus(){
        var tag = document.getElementById("i1");
        if(tag.value=="请输入关键字"){
            tag.value="";
        }
    }
    function Blur() {
        var tag = document.getElementById("i1");
        var tagvalue = tag.value;
        if(tagvalue.trim().length==0){
            tag.value="请输入关键字";
        }
    }
</script>
</body>
```
